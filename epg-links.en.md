# EPG links for DACH IPTV

## Recommended EPG sources

- Switzerland / many CH channels: `https://xmltv.ch/xmltv/xmltv-tvsd.xml.gz`
- `iptv-org` context: separate EPG project at `https://github.com/iptv-org/epg`

## Notes

- For Switzerland, `xmltv.ch` is explicitly mentioned in the context of `iptv-ch` for some provider playlists.
- For Germany and Austria there is no single, clearly documented universal EPG link in the referenced projects that reliably covers all channels.
- Therefore, the repository uses one central XMLTV link for CH; for DE/AT, additional per-channel EPG mapping inside your player is recommended.
